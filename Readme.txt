Command of Running this app:
node sim900.js --port "COMX" --sender_email "abc@gmail.com" --sender_username "abc" --receiver_email "xyz"

Arguments:
--port = Mention COM port number on which your modem is connected e.g "COM8"
--sender_email = Mention email address of sender e.g abc@gmail.com
--sender_username = Mention username of sender e.g abc
--receiver_email = Mention email address of receiver
